# CPW155-CRUD-App
.NET WinForms application to demonstrate CRUD (Create, Retrieve,
Update, Delete) operations with a SQL Server database 
using ADO.NET.

## Getting Started
- Install .NET 10
- Recommended IDE: VS 2026
- Clone the repo
- Install SQL Server developer with default settings
- Run `TennisDb.sql` against localhost
- Run the app
